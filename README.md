# hello-cv

はじめて Carbon Design System を使ってみます。

## 準備

```
% npm install -g @vue/cli
% vue create hello-cv
```
@vue/cli 3.7 じゃなくて 4.1.3だが、まあいいか。

```
% npm install @carbon/vue
```

指示通りに main.js と HelloWorld.vue を変更する。

```
% npm run serve
```
